Всего сделок  Эта форма предназначена для примерного расчета зрелости трейдера по его торговому отчету.

Успешность трейдера сравнивается с результатами заключения сделок по случайному закону, то есть, например, по результатам бросания костей обезьянкой, и далее взвешивается по количеству сделок.

Заполните поля формы значениями из торгового отчета трейдера и нажмите Рассчитать.

Альтернативная оценка дается в сравнении с обезьянкой, ибо обезьянка, согласно исследованиям, лучше 95% управляющих (впрочем, к этому советую отнестись с юмором).

Внимание! Результат расчета будет более приближен к реальному при задании спреда (спред можно указать только если известны результаты в пипсах).
